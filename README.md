# MemoCups

Juego casual mobile-first de memoria con tres vasos, hecho con React, Vite, TypeScript, Tailwind CSS, Framer Motion, Zustand y Howler.js.

## Comandos

```bash
npm install
npm run dev
npm run build
```

## PWA

La app ya incluye `manifest.webmanifest`, `theme-color` y un `sw.js` placeholder. Para convertirla en PWA completa, el siguiente paso es agregar iconos reales, estrategia de cache y prompt de instalacion.
